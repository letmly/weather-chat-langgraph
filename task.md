# **Тестовое задание: AI-чатбот с агентом.**

**Задачи:**

1. **Backend: LangGraph агент с инструментами**

Реализуй агента на LangGraph:

**Суть механики:** агент вызывает `get_weather(city)`, бэк возвращает не просто текст, а structured payload с полем `ui_event`. Фронт его читает и рендерит погодную карточку поверх чата — как нативный компонент, а не текст в пузыре.

Флоу

```
Пользователь пишет в чат
"Какая погода в Берлине?" — обычное сообщение, POST /chat с полем message.
V
LangGraph агент решает вызвать тул
LLM возвращает tool_call с именем get_weather и аргументом city="Berlin".
V
Tool node делает запрос к weather API
Вызывает OpenWeatherMap (или mock). Получает temp, condition, humidity, wind.
V
Бэк возвращает structured response backend
В ответе два поля: text (для LLM) и ui_event (для фронта). Фронт их разделяет.
V
Фронт рендерит weather card frontend
Читает ui_event.type === "weather_card", рендерит компонент рядом с текстовым ответом.
```

1. **Frontend: чат-интерфейс**

React (или Next.js) чат с:

- Полем ввода + кнопка отправки
- Историей сообщений (user / assistant)
- Индикатором загрузки пока агент думает

Стек и стайлинг — на усмотрение кандидата.

1. **Состояние и память (бонус)**

Хранить историю сообщений в state LangGraph-а и передавать в контекст LLM. Агент должен помнить предыдущие реплики в рамках сессии.
